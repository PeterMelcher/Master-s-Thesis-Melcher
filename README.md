The directory is grouped in the following folders:

* Pre-Processing
* Meshing
* item Simulation
* Post-Processing



The script used to create the lid of the watertight surface in Blender is located in the Pre-Processing folder.

The Meshing folder contains the relevant files for generating the meshes.

Similarly, the 0 and system folder files relevant for the simulation are located in Simulation.

Finally, the Post-Processing folder features the topSetDict and the calc\_distance.py file to generate a spatial y\_p distribution; the functions implemented in the controlDict, as well as the autopsy.py, plot\_stability.py, and simulation\_time.py scripts used to directly evaluate simulations.
